# .NET Core Buildpack for Heroku
### For .Net Core LTS Versions 3.1 to Current
## by Softtrends LLC

This Buidpack can be used to compile and deploy .Net Core application, ASP.Net Application, ASP.Net MVC Application to Heroku. It will pull the .NET Core dependencies from Microsoft, build a .NET Core Application and deploy it to the Heroku Platform. You should use Visual Studio 2019/2022 for best compatibility. Any tool from Visual Studio Code to the fully-featured Visual Studio Enterprise is supported.
